ENVIRONMENT MANAGEMENT
================================================================

CONDA ENVIRONMENT BASICS:
----------------------------------------------------------------
Check current environment:
  echo %CONDA_DEFAULT_ENV%

List all environments:
  conda env list

Activate AI2025 environment:
  conda activate AI2025

Deactivate current environment:
  conda deactivate

PACKAGE MANAGEMENT:
----------------------------------------------------------------
List installed packages:
  conda list
  pip list

Install packages:
  conda install package-name
  pip install package-name

Update packages:
  conda update package-name
  pip install --upgrade package-name

Remove packages:
  conda remove package-name
  pip uninstall package-name

TROUBLESHOOTING:
----------------------------------------------------------------

Problem: Wrong Python version
Solution:
  1. Check environment: echo %CONDA_DEFAULT_ENV%
  2. Should show: AI2025
  3. If not: conda activate AI2025

Problem: Package not found
Solution:
  1. Check if in correct environment: conda list
  2. Install if missing: pip install package-name
  3. Try alternative: conda install package-name

Problem: Import errors
Solution:
  1. Verify package installed: pip show package-name
  2. Check Python path: python -c "import sys; print^(sys.path^)"
  3. Reinstall if needed: pip install --force-reinstall package-name
